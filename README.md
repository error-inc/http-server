# HTTP Server in C v0.0

### Hi this is a simple http server in 'C'. It uses TCP/IP under the hood. When a client connects to the server, server sends a "Hello, World!" message to the clients as http response.

## Installation:

### Linux:
>You can get C source file and compile it yourself. Or Get already available binary 'server'  
>port: 8080

### Windows
>Not supported Yet!
